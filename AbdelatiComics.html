/ abdelati comics: your gateway to the world of comics

Engineered to provide comic book enthusiasts with a seamless shopping experience, Abdelati Comics combines an intuitive interface with a robust backend, ensuring a smooth integration of product displays and transactional operations.

/ project overview

The Abdelati Comics platform is designed to cater to both casual and dedicated comic book readers, offering a wide range of products

/ features

  • An extensive catalog of comic books and related merchandise
  • User-friendly shopping cart and secure checkout processes
  • Personalized recommendations based on user preferences and purchase history
  • Membership options for exclusive deals and early access to new releases

/ technologies used

  • HTML and CSS for a responsive frontend
  • Python for server-side logic
  • SQL databases for managing user data and inventory

/ challenges and learning

Developing a responsive and secure e-commerce platform posed challenges in handling real-time inventory updates and securing user transactions, which were addressed through robust backend logic and comprehensive testing.

This project enhanced our understanding of e-commerce systems and refined our skills in integrating various frontend and backend technologies.
